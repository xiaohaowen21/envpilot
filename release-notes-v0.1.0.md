## English

The first public release of EnvPilot, focused on Windows runtime installation, version switching, environment variable governance, and virtualization capability checks.

This release includes:

- clearer runtime installation stages across download, install, and environment refresh
- safer PATH synchronization without clobbering machine-level command access
- improved UI hierarchy for software inventory, runtime management, and variable governance
- a rewritten bilingual README for the open-source repository

This project is still in an early stage. Please read KNOWN_ISSUES.md before testing on a real machine.
